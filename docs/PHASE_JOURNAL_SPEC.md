# PHASE_JOURNAL_SPEC.md — ジャーナル機能 設計仕様

> このファイルは `docs/` に置き、Claude Code セッション開始時に参照させること。
> ジャーナル関連の実装・修正・追加のたびに最初に読む仕様書。

---

## 概要

ユーザーが1日2回（朝・夜）気分スコアと一言メモを入力し、過去の記録をグラフと一覧で振り返れる機能。

---

## データモデル

| フィールド | 型 | 説明 |
|---|---|---|
| `date` | `string` (YYYY-MM-DD) | 記録日 |
| `timeOfDay` | `"morning" \| "evening"` | 朝 or 夜 |
| `mood` | `number` (1〜10 整数) | 気分スコア |
| `note` | `string` | 一言メモ（任意、空文字OK） |
| `cycleDay` | `number` (1〜7) | 7日サイクルの何日目か |
| `userId` | `string` | Cognito ユーザーID |

- 1日のエントリは**朝・夜の最大2件**。同じ `date` + `timeOfDay` の組み合わせは1件のみ（上書き or 更新）。
- `cycleDay` は入力時点の `Character.cycleDay` から取得して保存する。

---

## 入力仕様

- **タイミング**: 朝（起床後）と夜（就寝前）の2回
- **気分スコア**: 1〜10 の整数。スライダーまたはタップ選択UI
- **メモ**: 任意の一言テキスト（文字数制限は100文字程度）
- 既に入力済みの場合は上書き更新できる（その日の記録を修正可能）

---

## 振り返り画面（`/journal/review`）

### 画面構成（上から順）

1. ヘッダー（タイトル + サブテキスト）
2. 統計ストリップ（横3カード）
3. グラフ / 一覧 タブ
4. タブコンテンツ

### 統計ストリップ

| カード | 内容 | 備考 |
|---|---|---|
| 期間平均 | 選択範囲内の全エントリの平均気分スコア（小数第1位） | 範囲切り替えで連動 |
| 連続記録 | 朝または夜どちらか入力があれば「記録あり」とカウントした連続日数 | 全期間で計算 |
| 最高スコア | 選択範囲内の最大 mood 値 | 範囲切り替えで連動 |

---

## グラフ仕様

### データ点の並び順

横軸は **朝→夜→朝→夜...** の時系列順。例：

```
6/23朝 → 6/23夜 → 6/24朝 → 6/24夜 → 6/25朝
```

### グラフ本数

**折れ線1本のみ。** 朝と夜を別線にしない。

### 点（dot）の見た目

| | 朝 | 夜 |
|---|---|---|
| 半径 | 小（3〜4px） | 大（4〜5px） |
| 色 | 薄い色（既存テーマの淡いアクセント） | 濃い色（既存テーマのメインアクセント） |

> **重要制約: 点の色はスコアの高低で変えない。**
> 朝か夜かの区別だけを色で表現する。
> 「高スコア=良い色」「低スコア=悪い色」という評価感をグラフに持ち込まない方針。
> （例: 朝=薄紫、夜=濃紫 のように固定。スコア5と8で同じ色になる。）

### Y軸

- `domain={[1, 10]}`
- 目盛りは `[2, 5, 8, 10]` 程度に絞る
- `y=5` に破線のリファレンスライン（基準線）

### X軸ラベルの間引き

**朝エントリの下にのみ日付ラベルを表示**（夜エントリの下は非表示）。
さらに範囲に応じて以下の間隔で間引く（読みやすさのため）:

| 表示範囲 | 日付ラベルの表示間隔 |
|---|---|
| 1週間 | 2日おき |
| 2週間 | 3日おき |
| 1ヶ月 | 7日おき |

結果として `6/19`, `6/22`, `6/25` のように適度な間隔で表示される。

### 凡例

グラフカード内に小さく朝/夜の点の意味を示す。文字は最小限。

### ツールチップ

タップ/ホバーで該当エントリの「日付・朝夜・スコア・メモ冒頭」を表示。

### 空状態

記録が0件のときは「まだ記録がありません」メッセージ + 入力画面への導線ボタン。

---

## 範囲セレクター

| ラベル | 日数 |
|---|---|
| 1週間 | 7日 |
| 2週間 | 14日 |
| 1ヶ月 | 31日 |

- グラフカードの**内側下部**に配置（グラフのすぐ下、カード外ではなく内側）
- 一覧タブにも同じセレクターを表示し、グラフと範囲を**共有**する（どちらで切り替えても連動）
- 統計ストリップの数値も選択範囲に連動する

---

## エントリピッカー（グラフタブ下部）

- 横スクロールできる朝夜ボタンの列
- ☀️（朝）/ 🌙（夜）アイコンで区別
- タップすると下の「選択中エントリ詳細カード」が切り替わる

---

## 選択中エントリ詳細カード

- 日付・朝夜アイコン
- 気分スコア（大きく表示）＋ 絵文字
- Day バッジ（`Day 3` など、cycleDay を表示）
- 一言メモ全文

---

## 一覧タブ

- 日付ごとにカードをまとめる（1日 = 1カード、その中に朝・夜のエントリ）
- 新しい日が上に来る（降順）
- 朝エントリには黄系の左ボーダー、夜エントリには青紫系の左ボーダー
- メモは2行で省略（全文は詳細カードで）

---

## 実装上の注意

- `recharts` を使用（`LineChart` + `Line` + カスタム `dot` + カスタム `XAxis tick`）
- データ取得は AmplifyClient の `list` で過去1ヶ月分を取得し、フロントでフィルタ
- ページネーション: DynamoDB の `list()` は最大100件。1ヶ月31日×2=62件なので通常は1リクエストで収まる。将来的に長期化した場合は対応要
- SSR不可（認証ユーザーデータのため `"use client"` 必須）
